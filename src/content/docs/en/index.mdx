## Features

<CardGrid stagger>
  <Card title="Tri-rate Decoupled Architecture" icon="rocket">
    Control 50 Hz / Physics 1 kHz / Render 7.5 Hz independently decoupled. Maximizes RL training throughput.
  </Card>
  <Card title="O(1) Environment Reset" icon="refresh">
    Snapshot-based Reset() without rebuilding the SOFA scene. Drastically reduces RL episode boundary overhead.
  </Card>
  <Card title="Multi-backend Compatible" icon="puzzle">
    `BackendABC` unified interface. Plug-and-play switch between SOFA / Blender / Mock. CI uses Mock, no GPU needed.
  </Card>
  <Card title="VLA / RL Native" icon="laptop">
    Gymnasium interface + ModelBridge adapter. Direct integration with TTT-VLA, Diffusion Policy and more.
  </Card>
</CardGrid>
