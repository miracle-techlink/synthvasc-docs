SOFA (Simulation Open Framework Architecture) 是 SynthVasc 的主要物理后端，
提供有限元软体仿真能力，用于导丝/导管的弯曲变形和血管壁接触力计算。

<Aside type="caution" title="需要 gpu-server 环境">
  SOFA 编译在 **gpu-server (172.30.1.53)** 上进行，需先通过 UniVPN 连接内网。
  本机 `conda env: sofa` 环境已预配置 SofaPython3 路径，但需要服务器端的共享库。
</Aside>

## 悬臂梁验证（Week 1 目标）

SOFA 后端的第一个物理验证场景：BeamFEM 悬臂梁受重力弯曲，对比解析解。

```python
# Week 1 验证脚本（开发中）
from synthvasc.backends.sofa import SofaBackend
from synthvasc.tests.fixtures import cantilever_config

backend = SofaBackend()
backend.initialize(cantilever_config)

for _ in range(100):
    state = backend.step(dt=0.001)

tip_deflection = state.positions[-1, 2]
analytical = cantilever_analytical_deflection(cantilever_config)
assert abs(tip_deflection - analytical) / analytical < 0.01, "FEM 误差 > 1%"
print(f"Tip deflection: {tip_deflection:.4f} m (analytical: {analytical:.4f} m)")
```
