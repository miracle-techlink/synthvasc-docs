## 核心模块

<CardGrid stagger>
  <Card title="CT → 合成 DSA" icon="document">
    Beer-Lambert 双平面 DSA 渲染，自动将任意增强 CT 转化为带 3D 标注的合成荧光序列，无需真实患者数据。
  </Card>
  <Card title="stEVE 仿真环境" icon="rocket">
    基于 SOFA BeamAdapter FEM 的物理精确导丝-血管交互仿真，Gymnasium 接口，支持 SAC / PPO / TD3。
  </Card>
  <Card title="虚实域迁移" icon="puzzle">
    带导丝保留损失 L_wire 的 CUT 无配对翻译，将合成 DSA 风格迁移至真实 guide3d 荧光图像分布。
  </Card>
  <Card title="Diffusion Policy 部署" icon="laptop">
    ResNet-18 + DDIM-10 离线训练，推理延迟 &lt;15ms/步，支持零样本迁移至物理机器人平台。
  </Card>
</CardGrid>

## 已验证性能

| 指标 | 数值 | 说明 |
|------|------|------|
| 导航成功率 | ~60% | SAC, multi-10 血管训练 |
| 推理延迟 | &lt;15 ms/步 | Diffusion Policy, DDIM-10 |
| 合成 DSA 帧数 | ~2,200 帧 | 11 例 KiTS AVT CT |
| SOFA 仿真速度 | ~1× 实时 | BeamAdapter FEM |
| 单次 CT 处理 | 10–15 分钟 | CPU，TotalSegmentator |
