AVT (Aortic Vessel Tree) 数据集包含主动脉及其分支的 CT 扫描与手工分割标注，
SynthVasc 使用其中 **27 个有效病例**作为仿真场景的血管网格来源。

<Aside type="tip" title="预处理完成">
  全部 27 个病例已完成：CT 分割 → 表面网格 → 中心线提取 → 导航点生成。
</Aside>

## 预处理流程

```
CT DICOM → TotalSegmentator 分割 → 血管 STL → MeshFix 修复
    → vmtk 中心线提取 → 导航路径点 NPY → SynthVasc Scene 导入
```
